Rails Application Template Projects h4. by Daniel Kehoe _Last updated 28 May 2014_ Rails application templates are Ruby scripts used with the @"rails new myapp -m"@ command to generate a Rails web application. They are often used to generate starter apps that can be the basis of any new Rails project. Also see an annotated list of "Rails Examples, Tutorials, and Starter Apps":http://railsapps.github.io/rails-examples-tutorials.html. h4. If You Are New to Rails If you're new to Rails, see "What is Ruby on Rails?":http://railsapps.github.io/what-is-ruby-rails.html, the book "Learn Ruby on Rails":http://learn-rails.com/learn-ruby-on-rails.html, and recommendations for a "Rails tutorial":https://tutorials.railsapps.org/rails-tutorial. h4. !http://railsapps.github.io/images/join/join-railsapps.png(Join RailsApps)!:http://railsapps.github.io/ h4. What is the RailsApps Project? This is an article from the RailsApps project. The "RailsApps project":http://railsapps.github.io/ provides example applications that developers use as starter apps. Hundreds of developers use the apps, report problems as they arise, and propose solutions. Rails changes frequently; each application is known to work and serves as your personal "reference implementation." Each application is accompanied by a tutorial so there is no mystery code. Support for the project comes from subscribers. Please accept our invitation to "join the RailsApps project":http://railsapps.github.io/. h2. Older Projects For a ranking of some older application templates by popularity, see: "The Ruby Toolbox: Rails App Templates":http://ruby-toolbox.com/categories/rails_app_templates.html These projects were updated in 2011 and use Devise for authentication: * "greendog99/greendog-rails-template":https://github.com/greendog99/greendog-rails-template bq. Devise, RSpec, Factory Girl, jQuery, Haml, Sass, Compass, Git, rvm, HTML5 Boilerplate, plus a layout. Uses subtemplates for easy customizing. * "dcrec1/rails3_template":https://github.com/dcrec1/rails3_template bq. Devise (with Cucumber steps), Cucumber, RSpec, Factory Girl, Haml, Compass, jQuery, Git, Formtastic, a layout, and much more. Uses subtemplates for easy customizing. * "aentos/rails3-templates":https://github.com/aentos/rails3-templates bq. Devise or Authlogic or OmniAuth, Cucumber, Capybara, RSpec, Factory Girl, Shoulda, jQuery, Haml, Compass, 960 grid or Blueprint, Capistrano or Heroku or Inploy, Git, plus a layout. Uses subtemplates for easy customizing. * "rubyx/rails3_template":https://github.com/rubyx/rails3_template bq. PostgreSQL, Devise, Cucumber with Selenium and Capybara, RSpec with Machinist and Faker, JQuery, Simple Form, Haml, SASS, Git, rvm, Google analytics (optional), Hoptoad for issue tracking (optional), TellThemWhen for downtime notifications (optional). With a description of the recipe in the README. These projects were updated in 2010 and use Devise: * "kenton/rails3-templates":https://github.com/kenton/rails3-templates bq. Devise, RSpec, Cucumber with Capybara, Factory Girl, jQuery, Simple Form, Git, rvm. Uses subtemplates for easy customizing. * "graemenelson/rails3-template":https://github.com/graemenelson/rails3-template bq. Devise, Cucumber, Shoulda with Factory Girl and Mocha, Haml, Sass, Formtastic, jQuery, Git, rvm. This project was updated in 2010 and uses Mongoid: * "madebydna/rails-templater":https://github.com/madebydna/rails-templater bq. Mongoid or ActiveRecord, RSpec, FactoryGirl, Cucumber (with Mongoid steps), jQuery, Haml, Sass, Compass. Uses subtemplates for easy customizing. These projects were updated in 2011 and do not use Devise or Mongoid: * "perfectline/template-bucket":https://github.com/perfectline/template-bucket bq. Cucumber, RSpec, jQuery, Haml or Jammit, Capistrano, Git, rvm, plus a layout. Uses subtemplates for easy customizing. * "leshill/rails3-app":https://github.com/leshill/rails3-app bq. Cucumber, RSpec, Factory Girl or Fabrication, Haml, jQuery, Git, rvm. * "shawn/shawns-rails3-template":https://github.com/shawn/shawns-rails3-template/ bq. Haml, Sass, Git, plus a layout. These Rails 2.3 projects were once popular but haven't been updated since 2009 or 2010: * "jm/rails-templates":https://github.com/jm/rails-templates * "ffmike/BigOldRailsTemplate":https://github.com/ffmike/BigOldRailsTemplate
