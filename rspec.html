Is It for You?

RSpec is the most popular test framework for Rails. This guide shows how to set up RSpec and explains how to write simple tests. It provides an introduction to basic concepts and recommended practices. RSpec is complex and there are very few comprehensive introductions available elsewhere, so experienced Rails developers as well as beginners will find this guide helpful.
