Xcode Command Line Tools

by Daniel Kehoe

Last updated 14 August 2013

How to install Apple Xcode Command Line Tools to get a GCC compiler for Mac OS X 10.8 Mountain Lion.

A C language compiler is not provided as part of the standard consumer installation of Mac OS X. Installation of many common Unix-based tools requires the GCC compiler. Xcode is a large suite of software development tools and libraries from Apple. Part of XCode, the Xcode Command Line Tools, includes a GCC compiler. This article shows how to install Xcode Command Line Tools.

Upgrade Mac OS X

To install the newest Xcode Command Line Tools, you’ll need the latest version of Mac OS X. If you’ve owned your Mac for several years and haven’t updated Mac OS X, be prepared to spend several hours updating the operating system.

See Apple’s instructions How to upgrade to OS X Mountain Lion. You can install Mac OS X 10.8 (Mountain Lion) from the Mac App Store for $19.99 (US). The upgrade is not available in Apple retail stores. Allow plenty of time for the download and installation, especially if you are several versions behind the current release.

Download

Xcode Command Line Tools is available as a standalone package (171 MB) or can be installed as part of the full Xcode suite (1.65 GB). Register as an Apple developer for the faster download.

Become an Apple Developer

For the faster download, register as an Apple Developer and download the standalone Xcode Command Line Tools from the Apple developer site. There is no charge to register as a developer, though you’ll be asked for marketing information. As a registered developer, you can download the standalone Xcode Command Line Tools without the full Xcode suite.

To register as an Apple Developer, visit the Apple Developer Site and fill out the registration forms. After registering, visit the Downloads page and search for “command line tools.” Look for the package “Command Line Tools (OS X Mountain Lion) for Xcode.” Click to expand the listing. Then click the file icon to begin the download.

After you’ve downloaded the file from the Apple Developer site, double-click the file to open a window. Double-click to install the Command Line Tools package.

Or Use the Mac App Store

Alternatively, Xcode can be installed directly from the Mac App Store. At 1.65 GB, it is a huge download and requires either a fast broadband connection or a great deal of patience.

If you don’t want to register as an Apple developer, use this link: Xcode on the Mac App Store. Click on “View in Mac App Store” to open the Mac App Store application. Click the “Install” button to begin the download. After installation from the Mac App Store, launch Xcode and go to “Preferences.” Under the “Downloads” tab, select and install “Command Line Tools.”

Alternatives

The OSX GCC Installer is an open source project to provide the GCC compiler and related tools. In a recent blog post, Xcode, GCC, and Homebrew, the project maintainer recommends installing the Xcode Command Line Tools from the Apple developer site because the open source project is unable to duplicate the complete Apple package (notably, the Node software project isn’t supported). If you can, install the Apple package.

Terminal Application

After downloading and installing Xcode Command Line Tools, use the Mac OS X Terminal application to verify the installation.

Find the Mac OS X Terminal application by using the Command-Spacebar combination and searching for “Terminal.” It’s in the Applications/Utilities/ folder.

The Terminal application or console gives us access to the Unix command line, or shell. We call the command line the shell because it is the outer layer of the operating system’s internal mechanisms (which we call the kernel).

Try out the terminal application by entering a shell command:

$ whoami

Don’t type the $ character. The $ character is a cue that you should enter a shell command. This is a longtime convention that indicates you should enter a command in the terminal application. The Unix shell command whoami returns your username.

Verify the Installation

After installation, use the Terminal application to verify that the GCC compiler was installed.

In the Terminal application, enter:

$ gcc --version
i686-apple-darwin10-llvm-gcc-4.2 (GCC) 4.2.1

You’ll see the GCC version identifier.

Start Coding!

With the GCC compiler installed, you can install any Unix system tools you need for software development.

To learn more about Unix shell commands, read The Command Line Crash Course.

For an introduction to programming, read Chris Pine’s book Learn to Program.
